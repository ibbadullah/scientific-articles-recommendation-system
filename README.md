# Scientific Articles Recommendation System

A web application that lets scientists and researchers search for any article and receive related article recommendations generated by a machine learning algorithm.

## Overview

Users search for an article of interest, and the system returns a ranked list of similar or relevant articles. Recommendations are produced by an ML model that analyzes article content and relationships rather than relying on simple keyword matching.

## Features

- Search for any article in the database
- ML-driven article recommendations based on content similarity
- Separate user and admin access levels
- Admin dashboard for managing the platform
- Pre-seeded user and admin accounts for testing

## Access and Roles

The application has two roles with strict separation:

- **General users** can log in, search, and view recommendations. They cannot access the admin area.
- **Admins** can access the admin panel and dashboard. The admin area is restricted to admin accounts only.

## URLs

| Purpose | URL |
|---------|-----|
| User login | `http://127.0.0.1:8000/login/` |
| Admin login | `http://127.0.0.1:8000/admin-panel/180.1.1/` |

> Note: General users cannot access the admin area. Sample users and admins have already been added to the system.

## Getting Started

1. Clone the repository

   ```bash
   git clone https://github.com/ibbadullah/scientific-articles-recommendation-system.git
   cd scientific-articles-recommendation-system
   ```

2. Create and activate a virtual environment

   ```bash
   python -m venv venv
   source venv/bin/activate   # On Windows: venv\Scripts\activate
   ```

3. Install dependencies

   ```bash
   pip install -r requirements.txt
   ```

4. Apply database migrations

   ```bash
   python manage.py migrate
   ```

5. Run the development server

   ```bash
   python manage.py runserver
   ```

6. Open the app at `http://127.0.0.1:8000/login/`

## Tech Stack

- Backend: Python, Django web framework
- Machine Learning: recommendation algorithm for article similarity
- Database: configured in project settings
